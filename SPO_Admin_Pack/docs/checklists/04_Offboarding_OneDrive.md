# Checklist – Offboarding użytkownika (OneDrive / SPO)

- [ ] Zidentyfikuj managera / osobę przejmującą dane
- [ ] Zabezpiecz OneDrive użytkownika (czasowy dostęp admin/manager)
- [ ] Sprawdź udostępnienia zewnętrzne z OneDrive
- [ ] Zarchiwizuj krytyczne dane (zgodnie z polityką)
- [ ] Wyłącz konto / usuń licencje zgodnie z procesem HR/IT
- [ ] Retencja danych zgodna z Purview (jeśli używana)
