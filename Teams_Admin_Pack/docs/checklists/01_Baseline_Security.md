# Checklist – Baseline bezpieczeństwa (Teams)

## A. Tożsamość (Entra)
- [ ] MFA dla adminów (minimum) + CA dla ryzykownych logowań
- [ ] Kontrola gości (B2B): kto może zapraszać, proces akceptacji
- [ ] External access (federation) skonfigurowany świadomie (allow/deny)

## B. Teams admin center – org-wide
- [ ] Guest access: ON/OFF zgodnie z polityką firmy
- [ ] External access: allow/deny list zgodnie z wymaganiami
- [ ] Meeting settings: nagrania, anonimy, lobby (baseline)
- [ ] Messaging settings: edycja/usuwanie, interop (baseline)
- [ ] App permissions: allow list / blokady custom apps (baseline)

## C. Compliance / Audit (jeśli używane)
- [ ] Audyt włączony i testowany
- [ ] Retencja audytu zgodna z wymaganiami
- [ ] Playbook incydentów: kto/ gdzie/ jak sprawdza zdarzenia
