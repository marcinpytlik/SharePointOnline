# Checklist – Aplikacje i polityki (Teams)

## A. Polityki (minimum do przeglądu)
- [ ] Meeting policy (nagrania, anonimy, lobby, transkrypcje)
- [ ] Messaging policy (czaty, edycja/usuwanie, link previews)
- [ ] Channels policy (private/shared)
- [ ] App permission policy (allow/deny)
- [ ] App setup policy (co jest przypięte)

## B. Aplikacje
- [ ] Lista dozwolonych aplikacji (baseline)
- [ ] Proces zatwierdzania aplikacji (security review)
- [ ] Blokada custom apps, jeśli organizacja tego wymaga
